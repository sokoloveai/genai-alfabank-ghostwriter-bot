# Ghostwriter Bot

Телеграм-бот генерирует посты для канала «Альфа Инвестиции». Сначала выберите рубрику, затем отправьте черновик текста или ссылку — бот соберёт итоговый пост в нужном стиле.

## Подготовка

1. Скопируйте файл `.env_example` в `.env` и заполните токен бота и ключ OpenAI.
2. Установите [uv](https://docs.astral.sh/uv/getting-started/).
3. Выполните `uv sync` для установки зависимостей.

## Локальный запуск

```bash
uv run python main.py
```

## Запуск в Docker

```bash
docker compose up --build
```

Бот отвечает на команды:
- `/start` — выбор канала и рубрики
- После выбора рубрики — любое сообщение или ссылка для генерации

Доступные рубрики канала `@alfa_investments`:
- `#АльфаИндекс`
- `#ЗанимательныеИнвестиции`
- `#ЧтоКупить`
- `#ГлавноеЗаНеделю`

Каждый запрос дополняется свежими примерами по выбранному хэштегу (few-shot). Можно добавить строку `тема: ...`, чтобы указать желаемый акцент.
